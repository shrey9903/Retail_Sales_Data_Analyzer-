# Retail Sales Data Analyzer

## Project Overview

The **Retail Sales Data Analyzer** is a Python-based data analysis project that processes and analyzes retail sales data from a CSV file. The program uses **Pandas, NumPy, Matplotlib, and Seaborn** to perform data manipulation, statistical analysis, and data visualization.

This project demonstrates the use of **Object-Oriented Programming (OOP)** along with data science libraries to generate insights from sales data.

---

## Features

* Load and clean sales data from a CSV file
* Perform **data manipulation and preprocessing**
* Calculate key sales metrics using **NumPy**
* Filter data based on **user-defined conditions**
* Generate dataset summaries
* Visualize sales insights using:

  * Bar Chart (Revenue by Product)
  * Line Chart (Daily Sales Trend)
  * Heatmap (Correlation Analysis)

---

## Technologies Used

* **Python**
* **Pandas** – Data manipulation and analysis

* **Matplotlib** – Data visualization

## Dataset Description

The dataset contains retail transaction data with the following columns:

| Column        | Description             |
| ------------- | ----------------------- |
| date          | Date of the sale        |
| product       | Product name            |
| category      | Product category        |
| price         | Price of the product    |
| quantity_sold | Number of units sold    |
| total_sales   | Total revenue generated |
